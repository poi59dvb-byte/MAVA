# Neon Rush (Vite + TypeScript + Three.js)

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Stack
- Vite + TypeScript
- Three.js (WebGL)
- DOM HUD/Menu UI (no React/Vue)

## Estrutura
A estrutura segue o layout pedido em `src/core`, `src/game`, `src/render`, `src/world`, `src/entities`, `src/ui`, `src/data`, `src/utils`.
